Метод `tasks.api.scrum.epic.get` возвращает значения полей эпика по его идентификатору.

## Параметры

#|
|| **Параметр** / **Тип** | **Описание** ||
|| **epicId**
[`unknown`](../../../data-types.md) | Идентификатор эпика. ||
|#
