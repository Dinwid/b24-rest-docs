## Описание

```http
catalog.productPropertyEnum.delete(id)
```

Метод удаляет значения списочных свойств.
Если операция успешна, возвращается `true` в теле ответа.

## Параметры

#|
|| **Параметр** | **Описание** ||
|| **id** 
[`integer`](../../data-types.md)| Идентификатор значения списочного свойства. ||
|#
