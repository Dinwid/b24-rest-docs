> Scope: [`bizproc`](../../scopes/permissions.md)
>
> Кто может выполнять метод: администратор

Метод возвращает список роботов, зарегистрированных приложением.

## Примеры

```javascript
BX24.callMethod(
	'bizproc.robot.list',
	{},
	function(result)
	{
		if(result.error())
			alert("Error: " + result.error());
		else
			alert("Успешно: " + result.data().join(', '));
	}
);
```
